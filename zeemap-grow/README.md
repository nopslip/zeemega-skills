# zeemap-grow

Grow a single zee into a leaf zee that builds on it. Part of the
[zeemega-skills](../README.md) pack.

See [SKILL.md](./SKILL.md) for full docs.

## Install

```
$HERMES_HOME/skills/productivity/zeemap-grow/   ← contents of this dir
```

Depends on `zeemap/lib/store.py` and `zeemap/lib/write_zee.py` —
install `zeemap/` from the same release.
